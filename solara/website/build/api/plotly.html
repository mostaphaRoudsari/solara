Renders a plotly figure object and support for events.

You can use a regular plotly figure, for instance using plotly.express.

Supported events are include in the example below.

For performance, a dependency list can be provided. Any change to the dependency list will cause the Figure data to be updated. This can be significantly faster than comparing he figure data directly.
